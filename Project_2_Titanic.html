Titanic数据探索项目¶

背景介绍：¶

泰坦尼克号是一艘英国皇家邮轮，也是白星航运公司旗下的3艘奥林匹克级邮轮之一，在其服役时间是全世界最大的海上船舶。 1912年4月10日，泰坦尼克号展开首航，也是唯一一次的载客出航，最终目的地为纽约。当时世界上最富有的一些人都是船上乘客，以及许多来自英国、爱尔兰、斯堪地那维亚和整个欧洲其他地区的移民，他们寻求在美国展开新生活的机会。 4月14至15日子夜前后，在中途发生碰撞冰山后沉没的严重灾难。 2,224名船上人员中有1,514人罹难，成为近代史上最严重的和平时期船难。船长爱德华·约翰·史密斯最终与船一起沉没；泰坦尼克号总设计师汤玛斯·安德鲁斯也在这起灾难中死亡。

数据来源¶

数据集来自 Kaggle 网站，包括泰坦尼克号上 2224 名乘客和船员中 891 名的人口学数据和乘客基本信息。本次研究将主要针对生还率展开，围绕甲板位置，乘客年龄，乘客富有程度（采用客运费作为评价指标）以及乘客性别与生还率之间的联系进行。

提出问题¶

  • 问题1：乘客的性别与生还率有怎样的关系？
  • 问题2：乘客的年龄与生还率有怎样的关系？
  • 问题3：乘客所在的甲板位置与生还率有怎样的关系？
  • 问题4：乘客的客运费水平与生还率有怎样的关系？
  • 问题5：在同一块甲板上的乘客中，生还率与乘客的客运费水平，有怎样的关系？

def group_age(df):
    age_cat=[]
    col_age=df.Age
    for x in col_age:
        if(x<1):
            age_cat.append('infant')
        elif(x<13):
            age_cat.append('child')
        elif(x<19):
            age_cat.append('teenager')
        elif(x<56):
            age_cat.append('adult')
        else:
            age_cat.append('the aged')
    return df.assign(Age_Cat = age_cat)

# 标准化客运费水平，增加为新的列，并增加客运费水平分类标签
def nor_group_fare(df, fare_col_name):
    fare = df[fare_col_name]
    fare_n = (fare-fare.mean())/fare.std()
    df = df.assign(Fare_N = fare_n)
    acf_cat=[]
    col_fare_n=df.Fare_N
    for x in col_fare_n:
        if(x<1):
            acf_cat.append('0_1')
        elif(x<2):
            acf_cat.append('1_2')
        elif(x<3):
            acf_cat.append('2_3')
        elif(x<4):
            acf_cat.append('3_4')
        else:
            acf_cat.append('>4')
    return df.assign(Fare_Cat = acf_cat)

def count_passenger(df, key):
    key_list = df[key].unique()
    new_df = pd.DataFrame({key: key_list})
    col_indexer = 'Passenger Count'
    for x in df[key]:
        row_indexer = new_df.index[new_df[key]==x]
        new_df.loc[row_indexer,col_indexer]=len(df[df[key] == x])
    return new_df

def create_df(df, key):
    key_list = df[key].unique()
    new_df = pd.DataFrame({'key': key_list})
    #df_pct = group_data(df, key)
    col_indexer = 'Percent'
    for x in key_list:
        row_indexer = new_df.index[new_df['key']==x]
        #new_df.loc[row_indexer,col_indexer]=df_pct.get(x)
        new_df.loc[row_indexer,col_indexer]=len(df[(df[key]==x) & (df['Survived']==1)])/len(df[df[key]==x])
    return new_df

我们怀疑这种差异是由性别差异或年龄引起（例如，0_1组包含了多数女性，或者多数幼者,他们受到了照顾，提高了小组的生还率）。由此展开进一步观察，对于同样的甲板，我们进行不同客运费水平分组的组间对比，以及0_1组的特别分析:

这样的结果部分符合我们之前的猜测：

  • 性别：
    • 0_1组男性比例44.44%，小组内部男性生还率为27.27% (全船的男性比例为64.8%，全船男性生还率为18.89%)
    • 2_3组男性生还率0%，唯一的男乘客遇难
    • '>4组男性生还率100%，远高于全船男性生还率与其他小组内部男性生还率
  • 年龄：
    • 0_1组的青少年有4人，孩童1人(合计占小组总人数的13.89%)，均生还 (全船青少年与孩童占总人数的18.5%，生还率42.86% 和 53.23%)

由对于0_1组的特别观察，性别的比例以及男性幼者受到的照顾影响了小组内部的生还率，显得高于全船乘客中客运费水平在0_1之间的生还率。

由组间对比，B甲板上的底层及中层的男性们的行为仍然是照顾把逃生机会让给幼童及女性，而最富有的阶层中的男性们选择了求生并且获得了成功。

以上观察的结果，由于受到样本量的局限，不适合用作代表全船整体的结论，只能够代表甲板B内部的情况。

得出结论¶

  • 由单个变量分析观察的结果，我们发现女性的生还率高于男性，婴孩与青少年的生还率高于成年人及老年人，不同的甲板上的乘客生还率差距较大（BDE明显高于 ACFG）,客运费水平的乘客群体的生还率明显低于较高客运费水平的乘客。
  • 由多维变量分析角度，单独取甲板B上的乘客群体观察，依旧是女性生还率高于男性，婴孩与青少年的生还率高于成年人及老年人，而客运费最低的乘客生还率低于客运费最高的乘客。

由以上发现，我们可以推测：¶

  • 在灾难降临时，女性，婴儿，孩童得到了更多的照顾，获得了更多求生的机会
  • 不同位置甲板的特性差别（例如沉没时间先后，与救生艇相隔的距离）导致了生还率的差异
  • 从贫富角度来看，处于最底层的乘客的生还情况整体弱于中层与上层阶级的乘客
  • 面对道德抉择（先让其他弱势群体逃生），小部分富有阶级得乘客会优先选择自己逃生

局限性：¶

  • 分组机制：单纯按客运费高低划分乘客的贫富程度，有一定的风险导致误差，例如没有考虑乘客旅程长短对于票价得影响。（但是，由于定价机制未知，且三港均在欧洲，尚未跨洋，所以我们可以假设不同港口上船的票价相差不会太多，故不再根据上船港口调整划分客运费水平。）
  • 变量间影响：对于每个单变量分析的前提假设了它与其他各变量独立，但是事实上未必如此。例如我们在观察年龄与生还率的关系时，假设全船乘客的其他各种因素（性别，贫富水平，所处甲板）因素均没有影响，但是事实上是有影响的，而且还有更多的潜在因素没有被选入本次研究的观察范围。
  • 样本量：在最后的基于B甲板的生还率观察，贫富-性别观察，贫富-年龄观察中，样本量只有数十人，所以最后的关于贫富阶级的不同选择的推测，也只对于B甲板内部情况有代表性。
